# Pokémon Go Data Visualization
<strong>EcoDex</strong> is an interactive website built using the D3.js library. The project visualizes data collected from the Pokémon Go app, allowing users to explore patterns of Pokémon appearance in relation to geographic and environmental factors.

## Features
Geospatial Visualization: Uses Hexbinning to display Pokémon population density on a global map.

Environmental Analysis: Detailed bar plots that show the correlation between appearance and specific conditions:

 - Time of Day

- Terrain Type

- Proximity to Water

- Weather

- Temperature

Dynamic Insights: View the stats of each Pokémon alongside a visual representation of their original images.

## Datasets used
The project integrates data from three different sources available on the Kaggle platform:

 - <a href="https://www.kaggle.com/datasets/hlrhegemony/pokemon-image-dataset?resource=download">Pokémon Image Dataset</a> – images of each Pokémon.

 - <a href="https://www.kaggle.com/datasets/semioniy/predictemall">Predict 'Em All</a> – information on locations and environmental conditions at the time of appearance.

 - <a href="https://www.kaggle.com/datasets/bretthammit/pokemongo-stats-dataset">Pokémon Go Stats</a> – attack, defense and pokemon stamina statistics.
